# perevodchik
Условие задания: Нам нужно реализовать программу, которая получает предложение на английском языке
(предложение состоит из подлежащего, сказуемого и дополнения; в нем в том числе присутствуют артикли “the”, “a”, “an”),
а после переводит его на русский язык. При этом, при наборе нескольких английских символов,на экран выводится подсказка 
с аналогом возможно предполагавшегося русского слова, все должно быть выполнено с использованием всплывающего окна.
